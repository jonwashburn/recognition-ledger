## Traditional Approaches to RH

### 1. Analytic Number Theory (Classical)
- **Method**: Study ζ(s) as complex function
- **Tools**: Contour integration, functional equation, Euler product
- **Progress**: Many partial results (zero-free regions, density theorems)
- **Limitation**: No clear reason why Re(s) = 1/2 is special

### 2. Spectral Theory (Hilbert-Pólya)
- **Method**: Find self-adjoint operator whose eigenvalues are zeros
- **Tools**: Quantum mechanics, random matrix theory
- **Progress**: Statistical evidence, GUE hypothesis
- **Limitation**: No natural operator found

### 3. Trace Formulas (Selberg, Weil)
- **Method**: Connect zeros to geometric/arithmetic data
- **Tools**: Automorphic forms, representation theory
- **Progress**: Deep connections revealed
- **Limitation**: Technical complexity obscures intuition

### 4. Probabilistic Approaches
- **Method**: Model primes as random variables
- **Tools**: Probability theory, random walks
- **Progress**: Heuristic arguments, Cramér's model
- **Limitation**: Primes aren't actually random

### 5. Physics-Inspired (Quantum Chaos, CFT)
- **Method**: Use physics intuition about quantum systems
- **Tools**: Path integrals, conformal field theory
- **Progress**: New perspectives, Berry-Keating
- **Limitation**: Still assumes primes as given
